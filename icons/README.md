# icons/

Place your custom PNG icons here.

Copilot will tell you exactly which icons to download for each diagram
at the top of the generated `my_diagram.py` as comments.

## Where to get free icons
- https://www.flaticon.com  (search any concept, download PNG 128px)
- https://icons8.com        (free PNG, huge library)
- https://www.svgrepo.com   (SVG → convert to PNG)

## Rules
- Format: PNG only
- Size: 64x64 or 128x128 px (recommended)
- Name: match exactly what Copilot uses in the script
  e.g. if script says `Custom("Shield", "./icons/shield.png")` → file must be `shield.png`
